Documentação do desenvolvimento do Jogo

HTML

Utilizei todo o poder do HTML5 para melhorar a semântica e a acessibilidade da página:

  • Novas TAGs
  • WAI-ARIA para tornar o código mais semântico, melhorando a integração com leitores de telas, por exemplo
  • Utilizei o Template Engine Jade para melhorar o desenvolvimento e organizar os arquivos HTML

CSS

Organização do CSS utilizando diversos conceitos, tais como:

  • Atomic Design
  • BEM
  • CSSOO
  • Pré-processador stylus para melhorar o desenvolvimento e organizar os arquivos CSS

Imagens

  • Utilizando o gulp, otimizei todas as imagens do projeto
  • Utilizei SVG
  • Utilizei Icon Fonts

JS

  • Incluí todo o JS em uma IIFE para manter o código em um escopo local
  • Organizei o código declarando as variáveis antes de tudo, também para otimizar a performance

Para saber mais sobre o Iceberg Boilerplate, que desenvolvi com Gulp, utilizado nesse projeto, acesse repositório no Github.
